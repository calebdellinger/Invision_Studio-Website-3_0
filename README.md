# Invision Creative — website

Photo-first **Invision Creative** site (dark, cinematic). The Next.js app lives in **`web/`**.

Partner brand: [Invision Marketing](https://invisionmarketing.io/) — cross-linked in the UI.

## Docs

- **[Site layout & IA](docs/WEBSITE_LAYOUT.md)** — routes, principles, chrome.
- **[AGENTS.md](AGENTS.md)** — agent and contributor orientation.

## Local development

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Point hosting **base directory** to `web`, then `npm run build` / Next adapter as required (e.g. Netlify/Vercel).
